## Descipción
Aplicacion de consola que permite guardar datos de los 
libros obtenida desde una API, guardarla en una base 
de datos y obtener los libros registrados en la base 
de dato propia


## Funcionalidades
1. Busca Titulo del libro y registra en la base de datos.
2. Listar libros registrados.
3. Listar autores registrados.
4. Listar autores vivos filtrando el año.
5. Listar libros registrado filtrado por idioma

## Tecnologías utilizada
- Java
- API
- Spring Boot
